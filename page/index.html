This page hosts the result of a survey gathering different software tools related to argumentation (CA). The focus is on tools and libraries that can be re-used for building CA-based applications. We will continuously update the survey so that researchers and potential practitioners can refer to it when selecting tools for their argumentation use cases.


In case of questions and feedback, contact us at tkampik[at]cs.umu.se

We update this table frequently, but you can also contribute by adding your argumentation software (tool, solver, visualizer, etc.) using either GitHub issues or this form.
